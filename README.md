This is my personal website that displays information about me. I made it as a part of Hackclub's Stardance but plan to continue using it in the future. It is hosted on Github Pages. It has 4 separate pages including Home, About Me, My Projects, and Reach out. Check it out here! https://arnaanshah.github.io/Personal-Website/

# How it's made
The website uses a combination of **HTML** and **CSS** along with a couple tricks to allow for a seamless look and some animations.

# Features
**My favorite features**
- Pixel art banana for the cursor
- Custom category tags for each project
- Working Contact Me form
- Animated scrolling project image bar

# Main Pages
1. The home page includes an image grid, a brief description of the website, some navigation buttons, and a most recent project section.
2. About me has information on who I am and what motivates me to be a maker.
3. The projects page has sections for each of my projects with tags showing what categories they fall into. It also has an automatically scrolling image bar that you can click on to be taken to a project.
4. The reach out page has a simple form to allow people to contact me if they need to. It asks for their full name, email, and a message and the data is securely stored.

# Style 
I used a dark, minimal scheme to try and keep the page simple but also included areas of color to make them stand out. I chose the main fonts because they remind me of the design company *Nothing.* I tried using a different layout or adding some variety where I could, but most of the pages look very similar. 

# Extras
This project took me a couple of days to make and I found it really fun. It was one of my first experiences with HTML and CSS aside from a small portion of a beginners computer science course. I loved being able to customize the website and make it look exactly how I want. I got a lot of help from this [video](https://www.youtube.com/watch?v=ZPMtug9qExk&t=899s) alongside resources like W3Schools and freeCodeCamp. I think what I could still add is more animations and reactions, more projects, and a better looking Reach Out page.
